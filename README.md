# CodeAlpha - CGPA Calculator

##  Project Overview

The CGPA Calculator is a C++ console-based application developed as part of the CodeAlpha C++ Programming Internship. This project allows students to calculate their GPA/CGPA by entering course grades and corresponding credit hours. The program computes total grade points, total credit hours, and displays the final CGPA accurately.


##  Objectives

* Implement fundamental C++ programming concepts.
* Perform calculations using user input.
* Apply loops, variables, and arithmetic operations.
* Develop a simple academic utility for students.

##  Features

* Input multiple courses.
* Enter grade points and credit hours for each course.
* Calculate total grade points.
* Calculate total credit hours.
* Compute and display final CGPA.
* User-friendly console interface.

##  Technologies Used
* C++
* Visual Studio Code
* GCC / MinGW Compiler

##  Program Workflow

1. Enter the number of courses.
2. Input grade points for each course.
3. Input credit hours for each course.
4. Program calculates:

   * Total Grade Points
   * Total Credit Hours
   * Final CGPA
5. Display the result on the screen.

##  CGPA Formula
CGPA = (Σ(Grade Point × Credit Hours)) / (Σ Credit Hours)

##  How to Run
1. Clone or download the repository.
2. Open the project in Visual Studio Code.
3. Compile the source file using a C++ compiler.
4. Run the executable.
5. Enter course details and view the calculated CGPA.
